CHAT
==============
A simple chat with the ability to create channels. The chat includes authentication and authorization.

### Installation

```console
$ git clone git@github.com:Valentina-Vasileva/frontend-project-12.git
$ cd frontend-project-12
$ make install
```

### Running locally

```console
$ make start
```
